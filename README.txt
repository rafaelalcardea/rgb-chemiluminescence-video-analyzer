Software for RGB analysis of chemiluminescence videos using circular ROI

This software was developed for processing chemiluminescence videos recorded with a smartphone. It allows batch video loading, fixed circular region-of-interest selection, manual ROI input, extraction of mean RGB signals, and peak detection for replicate injections.

Main features:
- batch processing of videos;
- fixed circular ROI;
- manual ROI input;
- extraction of R_mean, G_mean and B_mean;
- export of CSV files with temporal profiles and peak summaries.
